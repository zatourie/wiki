  * 새로 만들기 Ctrl+N
  * Quick Access Ctrl+3
  * 코드 자동완성 Ctrl+Space
  * import 정리 Ctrl+Shift+O
  * 빨리 수정 Ctrl+1
  * 한 줄 삭제 Ctrl+D
  * 코드 정렬 Ctrl+Shift+F


  * 선언한 곳으로 F3 또는 Ctrl+클릭
  * 이전 위치로 Alt+←
  * 파일 아웃라인 Ctrl+O
  * 상속 구조 Ctrl+T
  * 선택 문자 찾기 Ctrl+K
  * 행번호로 이동 Ctrl+L
  * 파일명으로 찾기 Ctrl+Shift+R
  * 프로젝트 텍스트 검색 Ctrl+H

  * 실행 Ctrl+F11
  * 단축키 목록 Ctrl+Shift+L
