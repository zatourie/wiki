# Functional Programming

발제자 : 이정훈

In C# (functional style):
```
result = mylist.Select(i => i + 1).Sum();
```

In C# (imperative style):
```
int sum = 0; 
foreach (int i in mylist)
{
   sum += (i + 1);
}
```
